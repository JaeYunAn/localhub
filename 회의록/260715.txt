회의록

1. 이슈공유
	- 다크 모드의 활용성 문제(적용 범위가 너무 좁음)
	- 챗봇 기능 문제, 챗봇 추가시 코드 오류
	- 가독성 문제(첫 화면이 너무 복잡함)
	- 커뮤니티 기능 문제(키워드 별로 필터가 안됨, 전체 게시글이 안보임, 게시글이 추가되었을때, 밑 부분이 짤림)
	- 홈페이지의 첫 화면으로 돌아가려면, 새로고침을 해야됨
2. 오늘 할 일(작업 범위)
	- 팀장 (안재윤) -> 배너 꾸미기, 카카오 지도 api 연결, 각각의 카테고리 마다의 상세정보 기능 구현
	- 팀원 (박재원) -> 커뮤니티 기능 구현(CRUD)
	- 팀원 (김재희) -> 챗봇 기능 구현
3. 개선사항
	- 다크 모드 적용 범위 설정 (상세 설명, 커뮤니티도 다크 모드 추가)
	- 챗봇 기능 다시 설정
	- 홉페이지의 이름을 누르면 첫 화면으로 돌아가게끔 만듦
