캘린더 기능
축제공연행사 버튼을 눌렀을 때 옆에 여행 기간을 입력하는 창이 뜸.
입력을 하면 해당 날짜에 해당하는 축제공연행사의 목록을 나열해줌.
거기에 해당되는 공연만 지도 위 핀포인트로 찍힌다.

캘린더 수정
1. 축제공연행사 버튼 바로 옆에 날짜 입력 창이 뜨게 해야함. O
2. 지정된 기간의 핀 포인트만 노출되게함.(기간내 해당하지 않는 핀포인트는 숨김) O
3. 커뮤니티 창에 기간에 해당하는 축제/행사 목록기능 삭제 O

검색 기능
검색창 출력 시기: 지역 버튼 눌렀을 때 출력 (메인 화면에서는 출력 안됨)
검색창 위치: 커뮤니티 열기/닫기 버튼 아래 (배너 내부에 존재, 지도 위로 침범 해선 안됨)
검색 시 해당하는 핀포인트만 지도 위에 출력

검색 수정
1. 검색하면 지도의 화면이 출력된 핀 포인터가 가운데로 오게 이동 O
2. 커뮤니티 열기/닫기 버튼이 다크 모드 전환 버튼과 검색창 사이에 오도록 이동 O

목록 기능
관광지, 레포츠, 문화시설, 쇼핑, 숙박, 여행코스, 축제 공연 행사 버튼을 누르면 커뮤니티 창 자리에 
목록창이 출력(커뮤니티 창이 종료되고 목록창이 출력되는 것임)
목록에서 컨텐츠를 클릭하면 지도 위에 해당하는 핀 포인트만 출력
(축제공연행사 목록은 축제공연행사 버튼을 눌렀을 시에 모든 축제공연 행사 목록이 출력 되다가 기간을 
설정하면 해당하는 핀포인트와 같이 기간에 해당하는 목록만 출력)
